# Practice (Python Basics)

Try writing code for each prompt. Start small, run often, and change one thing at a time.

## 1) Syntax basics
- Print your name and age.
- Use an `if/else` to check whether a number is even.

## 2) Data types
- Create variables of types: `int`, `float`, `str`, `bool`.
- Convert a string `"123"` into an integer and add `10`.

## 3) Operators
- Compute: `(a + b) * c` using variables.
- Use comparison operators to check if a student can access a system.

## 4) Strings
- Slice a string to get the first 3 characters.
- Replace one word in a sentence.
- Split a comma-separated string into a list.
